整体流程

  1. 主窗口->主进程: 打开子窗口
  2. 子窗口->主进程:初始化完成
  3. 主进程->子窗口:路由到指定页面
  4. 子窗口->主进程:页面加载完成
  5. 主进程->主窗口:子窗口已经可以使用了
  6. 窗口间可以互相通信

以上例子是在一个主窗口和一个子窗口的情况下进行的，如果存在多个窗口或者复杂的路由，可能需要把代码抽象封装的更合理。
